Installation

CDN (Quick Start)

The easiest way to get started is using a CDN:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/foxshack/simplegallery@1.0.3/dist/gallery.css">
<script src="https://cdn.jsdelivr.net/gh/foxshack/simplegallery@1.0.3/dist/gallery.js"></script>

NPM

Install via npm for use in build systems:

npm install -D github:foxshack/simplegallery#1.0

Then import in your JavaScript:

import simpleGallery from 'simplegallery';

// Initialize with default options
simpleGallery();

// Or customize the selector
simpleGallery({ selector: '.my-gallery' });

Usage

Wrap your images in a container with the .gallery class (or your custom selector):

<div class="gallery">
  <img src="./images/img1.jpg" alt="Malaga at Christmas">
  <img src="./images/img2.jpg" alt="Seagull">
  <img src="./images/img3.jpg" alt="Lorem ipsum">
</div>

<script src="./dist/gallery.js"></script>

Each image automatically gets a click handler to launch the modal gallery.

Data Attributes

Enhance your gallery with optional data attributes:

Attribute Description Example
data-sg-desc Description text displayed in the modal data-sg-desc="Beautiful sunset"
data-sg-src Alternate image source for modal (e.g., high-res version) data-sg-src="./images/img-hd.jpg"
data-sg-youtubeid YouTube video ID to embed instead of image in modal data-sg-youtubeid="dQw4w9WgXcQ"

Example with Data Attributes

<div class="gallery">
  <img src="./images/thumb1.jpg" 
       data-sg-src="./images/full1.jpg"
       data-sg-desc="This is my description" 
       alt="Image 1">
  <img src="./images/video-thumb.jpg" 
       data-sg-youtubeid="4uxAecTudqE" 
       alt="Video Title">
</div>

Customization

Simple Gallery uses CSS variables for easy theming. Override these in your stylesheet:

Variable Description Default
--sg-modal-bg Modal background color rgba(0, 0, 0, 0.95)
--sg-text-color Text color in modal #ffffff
--sg-btn-bg Button background color rgba(255, 255, 255, 0.2)
--sg-modal-zindex Modal z-index 1000
--sg-font-family Font family system-ui, sans-serif

Example Customization

:root {
  --sg-modal-bg: #e6e6e6;
  --sg-text-color: #3b3b3b;
  --sg-btn-bg: #b9c8c8;
  --sg-modal-zindex: 10000;
  --sg-font-family: Tahoma, sans-serif;
}
Note: The included CSS provides modal styling but not page layout for your gallery. You'll need to style the gallery container and images yourself to fit your design.
